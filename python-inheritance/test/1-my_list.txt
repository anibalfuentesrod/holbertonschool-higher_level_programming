====================
How to use 1-my_list
====================

THis module defines a class MyList tha inherits the built-in list:

test:
    >>> MyList = __import__('1-my_list').MyList

    >>> my_list = MyList()

    >>> my_list
    []

    >>> my_list.append(1)

    >>> my_list.append(4)

    >>> my_list.append(2)

    >>> my_list
    [1, 4, 2]

    >>> my_list.print_sorted()
    [1, 2, 4]

    >>> my_list
    [1, 4, 2]
